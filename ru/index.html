Обо мне

В 2022 году закончил ВО по направлению "Прикладная математика", в учебе применяли C# . Сейчас развиваюсь в направление backend разработки на C# - заканчиваю 12-месячные курсы. До ноября 2023 совмещал работу с учебой. Есть большое желание заняться разработкой веб-приложений.

На данный момент могу написать C# приложение в Visual Studio с CRUD функционалом на Entity Framework под MSQL Server, PostgreSQL, SQLite или другую БД, с использованием Git. Если говорить о БД - умею писать SQL запросы с join конструкциями

Есть фронт база - HTML, CSS, JS. Понимание DOM, погружения-всплытия событий, селекторов. Сейчас изучаю ASP.NET Core, MVC и WebApi
